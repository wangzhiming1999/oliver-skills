# Oliver Skill

前端开发相关的 Cursor / OpenClaw Skills 合集，可用于 ClawHub 上架或本地使用。

**作者**: [@wangzhiming1999](https://github.com/wangzhiming1999)  
**仓库**: https://github.com/wangzhiming1999/oliver-skill

---

## Skills 列表

| 目录 | 名称 | 说明 |
|------|------|------|
| bug-investigation | Bug 排查 | 可复现步骤与假设验证，快速定位前端 Bug 根因 |
| component-api-design | 组件与 API 设计 | 设计易用、可扩展的 React/Vue 组件 API 与文件结构 |
| design-to-code | 还原设计图 | 将 Figma/Sketch/图片高保真还原为前端代码 |
| modified-code-review | 修改代码评审 | 评审 diff/PR，最佳实践、性价比与代码评分 |
| refactor-safely | 安全重构 | 分步可验证重构，不改变对外行为 |
| frontend-performance | 前端性能优化 | LCP/FCP/CLS、打包体积与运行时优化方案 |

---

## 本地使用（Cursor）

将单个 skill 的目录复制到 Cursor 的 skills 目录，或在本仓库中通过 `.cursor/skills` 软链接指向 `oliver-skill/skills` 下的子目录。

## 上架 ClawHub

参见 [CLAWHUB_PUBLISH.md](./CLAWHUB_PUBLISH.md)，按步骤打包并提交到 https://clawhub.ai 。
